ASISTENCIA TECNICA GRATUITA PARA LOS CLIENTES

Asistimos a nuestros clientes con problemas técnicos o de cualquier tipo tengan con nuestros productos

SERVICIO DE REPARACION A DOMICILIO

Ofrecemos el servicio de reparacion a domicilio para nuestros productos, (el cliente puede especificar si enviar el producto a nuestra empresa o pedir la reparacion a domicilio)

SERVICIO DE VENTA DE PRODUCTOS POR PARTE DEL CLEINTE

El cliente puede vender productos propios en nuestra tienda, siempre y cuando los productos estén relacionados con la informática

SERVICIO DE COMPRAVENTA DE PRODUCTOS INFORMÁTICOS

Ofrecemos diversos productos de informática a nuestros clientes, además de vender sistemas operativos

MONTAJE DE EQUIPOS INFORMÁTICOS

Los clientes pueden montar sus propios equipos informáticos, ya sean PC de sobremesa o portatiles, ayudamos en la medida de lo posible a nuestros clientes a la hora de montar el producto, ya sea usando videos u ofreciendo el servicio de montar nosotros mismos el producto y enviarlo a domicilio.

SERVICIO DE INSTALACION DE SISTEMAS OPERATIVOS

Si compras todos los productos necesarios para montar un equipo y un sistemas operativo para instalarlo tras el montaje, puedes pedirnos que instalemos el sistema operativo por ti, o directamente que montemos todo el equipo incluyendo el sistema operativo
